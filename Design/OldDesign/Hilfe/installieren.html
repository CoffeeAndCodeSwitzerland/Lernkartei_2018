Lernkartei - Installation

"Lernkartei" wurde mit Hilfe der Programmiersprache JAVA erstellt. Somit läuft die "Lernkartei" auf Windows-, LINUX- und MAC-Rechnern.

Damit das Programm läuft, musst du auf deinem Rechner aber auch JAVA haben. Je nach Betriebssystem oder Konfiguration ist JAVA bereits installiert. Vielleicht weisst du nicht, ob das bei dir der Fall ist. Das macht nichts: installiere zuerst die "Lernkartei" und falls sie dann nicht läuft, kannst Du immer noch JAVA installieren.

Wenn du mit SW-Installieren keine Erfahrung hast, ist das nicht weiter schlimm.
Probiere es mit dieser Anleitung einfach mal aus. Wenn es nicht klappt, frage doch jemanden der mehr Erfahrung hat.
Ist das Programm erst mal installiert, ist die Bedienung ganz einfach.

Lernkartei von Internet herunterladen und installieren

Die Lernkartei kannst du vom Internet als komprimiertes Zip-Archiv herunterladen von http://www.zhwin.ch/~hrt/Lernkartei/Lernkartei.zip (ca. 500kB gross).
Erstelle ein Verzeichnis Lernkartei irgendwo auf deinem Rechner und dekomprimiere das Zip-Archiv in dieses Verzeichnis.

Lernkartei mit Doppleklick starten

Ein Doppelklick auf die Datei Lernkartei.jar startet die Lernkartei.

Falls es nicht klappt, gibt es zwei mögliche Ursachen:

  • JAVA ist noch nicht installiert: siehe Abschnitt "JAVA installieren"
  • JAVA ist installiert, aber der Datei-Typ "*.jar" ist nicht mit JAVA verknüpft: siehe Abschnitt "Lernkartei von der Kommandozeile aus starten"

Lernkartei von der Kommandozeile aus starten

Zuerst musst du ein Kommandozeilen-Fenster öffnen. (Dieses heisst "Eingabeaufforderung" unter Windows, "Shell" unter LINUX und "Terminal" unter "MAC OSX".)
Mit dem Befehl "cd" wechselst du ins Verzeichnis "Lernkartei". (Z.B. cd \Lernkartei (Windows) oder cd Lernkartei (LINUX und MAC) eintippen.)
Eintippen von java Lernkartei startet jetzt die Lernkartei. (Ev. muss der Pfad zu java explizit angegeben werden, z.B. so: C:\WINDOWS\java.exe Lernkartei .)

JAVA vom Internet herunterladen und installieren

  • Um JAVA herunterzuladen, gibst du http://java.com/de/download/ im Browser ein.
  • Klicke auf "Manuell herunterladen".
  • Klicke auf den "Download"-Knopf neben dem von Dir genutzen Betriebssystem.
  • Rechts vom "Download"-Knopf befindet sich ein Link mit den Installations-"Anweisungen".
  • Jetzt kannst Du die Lernkartei mit Doppelklick auf die Datei Lernkartei.jar starten (siehe oben).

Verknüpfungen auf dem Desktop

Um die Lernkartei direkt starten zu können, kannst du Verknüpfungen (Aliase) auf dem Dektop erstellen.

  • zur Datei Lernkartei.jar im Verzeichnis Lernkartei, um das Programm zu starten.
  • zur Datei Lernkartei.html im Unterverzeichnis Hilfe um die Hilfe anzuzeigen.
